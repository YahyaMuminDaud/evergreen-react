# Evergreen Matf LLC — Website

React website for Evergreen Matf LLC Commercial Cleaners.

## Getting Started

```bash
npm install
npm start
```

Opens at http://localhost:3000

## Build for Production / GitHub Pages

```bash
npm run build
```

Then push the `build/` folder to GitHub Pages.

## File Structure

```
src/
  App.js               — Router & page layout
  components/
    Nav.js / Nav.css   — Sticky navigation
    Footer.js / Footer.css — Footer
    PineSVG.js         — Reusable pine cone logo SVG
  pages/
    Home.js / Home.css       — Landing page
    Services.js / Services.css — Services & pricing
    WhyUs.js / WhyUs.css     — Why choose us
  styles/
    global.css         — Shared variables, buttons, sections
```
